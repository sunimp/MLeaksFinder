# MLeaksFinder

## Example

To run the example project, clone the repo, open `MLeaksFinder-Example.xcodeproj` file.

## Requirements

* Xcode 15.4+
* Swift 5.10+
* iOS 14.0+

## Installation

### SPM (Recommended):

Add dependency:
```swift
.package(url: "https://gitlab.webullbroker.com:5253/stocks/MLeaksFinder.git", from: "2.0.0")
```

### CocoaPods:

```ruby
pod 'MLeaksFinder', '>= 2.0.0'
```
