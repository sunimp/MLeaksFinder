# MLeaksFinder

## Example

To run the example project, clone the repo, open `MLeaksFinder-Example.xcodeproj` file.

## Requirements

* Xcode 15.4+
* Swift 5.10+
* iOS 14.0+

## Installation

### Swift Package Manager:

```swift
dependencies: [
    .package(url: "https://github.com/sunimp/MLeaksFinder.git", .upToNextMajor(from: "0.1.0"))
]
```
